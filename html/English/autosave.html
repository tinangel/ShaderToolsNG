Auto-backup :
You can set this parameter into 'Settings' element and save your files automatically :
The backup frenquency can be edit between 5 and 50 executions of Blender3D. The list of files saved are :
  • ShaderToolsApis : this file contains API's functions needed by ShaderTools.
  • ShaderToolsConfigs : this file contains the differents settings.
  • ShaderToolsDatabaseNG : this file is your materials database (WARNING: here it is the default name).
  • ShaderToolsLanguages : this file contains all languages supported.
