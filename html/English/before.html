- A computer running Blender3D (2.60.00 and next versions). Today almost all computers can execute Blender3D. However, I can provide you my hardware configuration used to create this add-on :
  • Intel Pentium (R) CPU 2.13GHz Dual Core P6200
  • 4GB of RAM
  • Graphics Card Mobile Intel IronLake
- You need 300MB to install it (the auto-backup can asked much space on your hard disk, please make sure you have enough space).
