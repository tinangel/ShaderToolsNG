Utilities :
That list allows to access more tools you could need. You must just click to an element :
  • Export or Import the materials database : this element allows you to export or import the database materials safely.
  • Open the path of the add-on : this element allows you to open work directory of ShaderTools.
  • Cleaning Tools : this element allows you to clean ShaderTools directories.
  • Migrate database (V1 -> V2) : this tool allows you migrate a database of materials from the old generation.
  • Credits : this tool allows you know contributors to this project.
  • Help : this tool allows you access to that help.
  • Log : this tool allows you to know state of operations performed.
  • New : this tool allows you to create a new material easily.
  • Settings : this tool allows you to create various configurations.
