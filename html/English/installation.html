Installation :
To install the ShaderTools, please follow the eleven steps (WARNING: I assume you have downloaded the zip archive, it is placed on your desktop and it is not unpacked):
STEP 1: Please execute Blender3D.
STEP 2: Select the 'File' menu and 'User Preferences ...' :
STEP 3: The window look like this :
STEP 4: select the 'Add-on' panel:
STEP 5: Click on button 'Install Add-On ...' :
STEP 6: In this new window, select location of the add-on. For our example it will be on the 'Desktop'. Then double-click on the file (you can also do a single click and then click on the button 'Install Add-On ...' at the top right corner of the screen) :
STEP 7: Comeback in the panel 'Addons' the ShaderTools should appear now alone. You need to check box to active it. (WARNING : Please must be patient because activation can be need few seconds)
STEP 8: To avoid having to redo all these steps make sure to check the button 'Save As Default':
STEP 9: If you are at this stage, congratulations, the worst is passed. Now you should be back to 3D view. Now you can go to the 'Material' panel (you can found it in the 'Properties' switch) :
STEP 10: Scroll the 'Material' panel and should have this window :
STEP 11: Restart Blender3D and go to the panel 'Material'. ShaderTools is now functional. (WARNING : if the addon is not updated please restart Blender3D and go back into the panel 'Material.)
