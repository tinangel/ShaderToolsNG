User interface :
Now we will see the user interface :
The user interface should look like that :
  • Open : this button allows you to access the materials database.
  • Save : this button allows you to backup the selected material into your materials database.
  • Export : this button allows you to export the selected material and to share him.
  • Import : this button allows you to import in your scene a material previously exported.
  • Utilities : this list allows you to access various tools that can be useful (see below).
  • Progess bar : indicate the state of progress of the various tools used.
The list of tools should look like this :
  • Export or Import the materials database : this element allows you to export or import the database materials safely.
  • Open the path of the add-on : this element allows you to open work directory of ShaderTools.
  • Cleaning Tools : this element allows you to clean ShaderTools directories.
  • Migrate a database (v1 -> v2) : this element allows you to migrate an old database created with ShaderTools (the database must be a version 0.9) to a recent database of the ShaderToolsNG (v2).
  • Credits : this element allows you to access the list of contributors project.
  • Help : this element allows you to access to this help.
  • Log : this element allows you to get informations about any errors occurred during use.
  • New : this element allows you to create a new material easily and using a common environment pre-visualization.
  • Settings : this element allows you to create differents needed configurations, including the possibility to manage multiple materials database.
