Settings :
This element allows to create multiple configurations you need.
Click on 'Utilities' into user interface and select 'Settings' :
A search window appears containing all configurations (here there is just one configuration and her name is 'Default') :
You can search into list using the search area (see example below). Now you must select a configuration you want use or modify to create a new one (WARNING : to create a new configuration you must use an existing) :
The window below must appears containing all informations to that configuration :
To create a new configuration you can edit configuration currently displayed (creator, description ....) :
Category allows you to organize your materials (this parameter affects the advanced search for example). You can use categories 'Personal' or 'Other' if others categories not correspond :
The validation action is very important because it will determine if you want 'Delete' or 'Register' settings. Our example we can remove 'Default' configuration or we can register our new configuration named 'My new configuration'. Therefore, the validation action will take effect when you press 'OK' button :
The preview settings allows configure size to the rendered preview when material is exported or when you save your material inside database (the resolution by default is 512x512).
When you use 'Export' button, the check box 'Create a preview' is activated. You can set that option here (activated or deactivated) :
WARNING : When you edit the database path, make sure you put the correct location of the new database otherwise ShaderTools will not be able to restart next time (but don't worry, if you wrong, you can remove file 'ShaderToolsConfigs.sqlite', you can find it into ShaderTools folder -> 'shader_tools_ng') :
The auto-backup allows manage frequency of backup on your computer (for more informations see section '- Auto-backup' ) :
To select a default configuration, you must check the box 'Default settings' and close window without click on 'OK' button (WARNING : it's the only option which can be activate or deactivate without click on 'OK').
Once your configuration completed just click on 'OK' (in our example we register) :
Your configuration should now appears in search window :
