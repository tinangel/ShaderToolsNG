Les Modules Logs, Credits et Aide :
Le module Logs vous permet d'obtenir des informations sur l'état de vos actions dans le ShaderTools.
Voici à quoi ressemble le fichier de Logs accessible à partir du module 'Utilitaires' :
  • Configs database exists : vérifie si le fichier de configuration existe.
  • Test to open a configuration : vérifie s'il est possible d'ouvrir une configuration (dans la cas contraire le ShaderTools restaurera la configuration par défaut ).
  • Materials database exists : vérifie si la base de matériaux existe.
  • Globals : le ShaderTools va vérifier plusieurs choses (elles sont représentées par des sous rubrique que l'on reconnait grâce aux différentes tabulations)
  • import : vérifie que les librairies Python sont correctement importées.
  • database exists : vérifie si la table de Test existe pour effectuer différents tests.
  • insert into database : test d'insertion de données dans la base.
  • select in database : test s'il est possible d'interroger la base.
  • update into database : test de mise à jour des données de la base.
  • default configuration verification : chargement de la configuration par défaut.
  • check folders : vérifie si les dossiers et sous dossiers sont correctement initialisés. Dans notre exemple nous pouvons voir que seul le dossier 'Francais' est 'ok', cela est normal puisque à l'heure où j'écris ce fichier d'aide seul la langue Française est prise en compte.
  • workbase : vérifie que le fichier de création d'une nouvelle matière existe bien.
  • help : vérifie que le fichier de sauvegarde de l'aide est accessible.
  • img : vérifie que le fichier de sauvegarde image d'erreur est accessible.
  • bookmark import : vérifie que les librairies Python d'import du module de création de bookmark sont bons.
  • bookmark save : fait une sauvegarde du fichier 'bookmark' de Blender3D avant d'effectuer la mise à jour.
  • bookmark folder : création du dossier 'Base de matériaux' sur le disque dur.
  • bookmark update : mise à jour du fichier 'bookmark' de Blender3D
  • Panel : vérifie si le panneau du ShaderTools ne contient pas d'erreurs.
