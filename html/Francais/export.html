Le Module Exporter :
Ce module vous permet d'exporter un matériau en vue de le partager.
Sélectionnez le matériau qui doit-être exporter puis cliquez sur le bouton 'Exporter' de l'interface du ShaderTools :
Une nouvelle fenêtre s'ouvre, dans le panneau de gauche saisissez les informations demandées (créateur ...):
Pour prévisualiser le matériau lié au fichier d'export (*.blex) vous pouvez ajoutez une prévisualisation. Pour cela placez-vous sur la case à cochée 'Creer une previsualisation' :
Cochez la case :
Vous devez donner un nom à votre matériau dans la zone supérieure de la fenêtre comme ici :
Puis vous devez sélectionnez le répertoire dans lequel votre matériau sera exporter (dans notre exemple c'est le bureau):
Puis vous devrez cliquez sur le bouton d'export situé en haut à droite de votre écran :
Enfin vous pouvez aller dans le répertoire où le materiau a été exporter (ici le bureau) et partager votre matériau :
