Le Module Configuration :
Ce module vous permet de créer de multiples configurations selon vos besoins.
Cliquez sur la liste des 'Utilitaires' de l'interface du ShaderTools puis 'Configuration' pour accéder à ce module :
Un liste de recherche répertorie l'ensemble de vos configurations (ici il y en a qu'une c'est celle par défaut dont le nom est 'Default') :
Vous pouvez effectuer une recherche dans la liste des configurations en utilisant la zone étudiée à cet effet comme dans cet exemple, il ne vous restera plus qu'à sélectionner la configuration que vous souhaitez utiliser ou modifier en vue d'en créer une nouvelle : (ATTENTION : pour créer une nouvelle configuration nous partirons d'une déjà existante)
Vous devriez voir apparaître la fenêtre suivante. Cette fenêtre contient toutes les informations relatives à la configuration précédemment sélectionner :
Pour créer une nouvelle configuration il vous suffira d'éditer la configuration comme bon vous semble (créateur, ...) :
Les catégories vous permettront de classer vos matériaux (ce qui influera sur le module de recherches avancés par exemple), à savoir que vous disposez d'une catégorie 'Personnel' et 'Divers' s'il vous est impossible de classer vos matériaux dans les autres rubriques :
L'action de validation est très importante car c'est elle qui va déterminer si vous souhaitez 'Supprimer' ou 'Enregistrer' la configuration sélectionnée précédemment. Dans notre exemple soit nous supprimerons la configuration 'Default' (c'est la configuration que nous avons sélectionner et qui porte le numéro 1) soit nous enregistrerons notre nouvelle configuration qui porte le nom 'Ma nouvelle configuration' (et qui prendra comme numéro 2); Par conséquent, suivant l'action de validation choisit, l'action ne prendra effet qu'au moment où vous appuierez sur le bouton 'OK' :
Les paramètres de prévisualisation déterminent la taille du rendu de la prévisualisation du matériau lorsque vous exporter un matériau ou encore lorsque vous sauvegarderez vos matériaux dans la base (par la suite lorsque vous utiliserez le module 'Ouvrir...' la qualité des matériaux qui apparaîtront en miniatures seront de plus ou moins bonne qualité suivant ces réglages. La résolution moyenne est de 512x512 pixels) :
Lorsque vous utilisez le module 'Exporter' par défaut la case 'Creer un previsualisation' est cochée. Il vous est possible de paramétrer cette option (en l'activant ou en la désactivant) comme ci-dessous :
ATTENTION : lorsque vous manipulez l'emplacement de la base de matériaux, assurez vous de bien renseigner l'emplacement de la nouvelle base de matériaux sous peine de ne pas pouvoir rédémarrer l'add-on au prochain lancement (cependant rassurez-vous, si vous deviez vous tromper, chose qui arrive à tout le monde, il vous suffira de supprimer le fichier 'ShaderToolsConfigs.sqlite' que vous trouverez dans le dossier d'installation du ShaderTools -> 'shader_tools_ng') :
L'auto-sauvegarde vous permettra de gérer la fréquence de la sauvegarde sur votre ordinateur, pour plus d'informations consultez la rubrique '- L'Auto-sauvegarde' du menu ci-contre :
Pour sélectionner une configuration par défaut il vous suffira de cocher la case 'Configuration par defaut' : (ATTENTION : c'est la seule option qui est active ou désactive sans que l'on ai besoin de cliquer sur le bouton 'OK'. Cette particularité réside dans le fait qu'elle vous permettra de basculer d'une configuration à l'autre sans avoir à recréer des configurations à l'infinie. Exemple : Vous disposez de deux bases de matériaux vous souhaitez basculer de la base 1 à la base 2, il vous suffira d'aller dans le module de configuration de sélectionner la configuration correspondant à la base 2 et de cocher la case 'Configuration par défaut' puis de sortir de la fenêtre de configuration sans à avoir à cliquer sur le bouton 'OK' qui je le rappelle 'Enregistre' ou 'Supprime' une configuration).
Une fois votre configuration créée il ne vous suffira plus qu'à valider en cliquant sur le bouton 'OK' (dans notre exemple on l'enregistre) :
