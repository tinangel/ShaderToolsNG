L'Interface :
Nous allons voir ici l'agencement de l'interface du ShaderTools :
Voici à quoi doit ressembler l'interface :
  • Ouvrir ... : ce module vous permet d'accéder à votre base de matériaux.
  • Sauvegarder : ce module vous permet de sauvegarder dans votre base le matétiau actuellement sélectionné.
  • Exporter : ce module vous permet d'exporter le matériau actuellement sélectionné en vue de le partager.
  • Importer : ce module vous permet d'importer dans votre scene un matériau préalablement Exporter.
  • Les utilitaires : ce module vous permet d'accéder à divers outils qui peuvent-être utiles (voir ci-dessous).
  • La barre de progession : indicateur sur l'état de la progression des divers modules utilisés.
La liste des utilitaires devrait ressembler à ceci :
  • Exporter ou Importer la base materiau : ce module vous permet d'exporter ou importer une base de materiaux en toute sécurité.
  • Ouvrir le dossier de l'add-on : ce module permet d'ouvrir le répertoire courant du ShaderTools.
  • Outils de nettoyage : ce module vous permet de nettoyer les différents répertoires du ShaderTools.
  • Migrer une base de données (v1 -> v2) : ce module vous permet de migrer une ancienne base créée avec le ShaderTools (la base doit-être une version 0.9 pour être migrée) vers une base ShaderToolsNG (v2).
  • Credits : ce module vous permet d'accéder à la liste des contibuteurs au projet.
  • Aide : ce module vous permet d'accéder à la présente aide via le ShaderTools.
  • Logs : ce module vous permet d'obtenir des informations sur les éventuelles erreurs qui pourrait survenir lors de l'utilisation de cet add-on.
  • Créer une nouvelle matière : ce module vous permet de créer un nouveau materiau trés simplement et en utilisant une pré-visualisation commune aux autres matériaux déjà existant.
  • Configuration : ce module vous permet de créer différentes configurations suivant vos besoins et notamment de gérer de multiples bases de matériaux.
