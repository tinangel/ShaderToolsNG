Le Module Utilitaires :
Ce module vous permet d'accéder à un ensemble d'outils suivants vos besoins. Il est accessible depuis l'interface du ShaderTools en faisant un simple clic comme ci-dessous :
  • Migrer une base de donnees (V1 -> V2) : cet outil permet de faire migrer une base de matériaux de l'ancienne génération du ShaderTools.
  • Credits : cet outil vous permet de connaître les différents contributeurs à ce projet.
  • Aide : cet outil vous permet d'accéder à l'aide via l'add-on.
  • Logs : cet outil vous permet de connaître l'état des différentes manipulations effectuées au cours de l'utilisation de l'add-on.
  • Creer une nouvelle matiere : cet outil vous permet de créer trés simplement un nouveau matériau.
  • Configuration : cet outil vous permet de créer diverses configurations suivant vos besoins.
