Les Prérequis :
Pour vous permettre de faire fonctionner le ShaderTools dans de bonnes conditions vous aurez besoin :
- d'un ordinateur capable de faire fonctionner Blender3D (au minimum la version 2.60.00). A savoir qu'actuellement quasiment tous les ordinateurs sont capables de faire fonctionner Blender3D. Cependant je peux vous fournir les caractéristiques que j'ai utilisé pour créer cet add-on :
  • Intel Pentium(R) CPU P6200 2.13Ghz Dual Core
  • 4Go de Mémoire Vive
  • Carte Graphique Intel IronLake Mobile
- un minimum de 100Mo pour l'installation de l'add-on seul (sachant que le système d'auto-sauvegarde et les sauvegardes de vos matériaux sont gourmands en espace, veillez à disposer de suffisamment de place pour une utilisation dans les meilleures conditions possibles)
Et c'est tout il me semble :)
