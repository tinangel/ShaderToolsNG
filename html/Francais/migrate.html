Le Module Migrer une base de donnees (V1 -> V2) :
Ce module vous permet de faire migrer votre ancienne base de matériaux du ShaderTools vers la nouvelle génération du ShaderTools NG : (ATTENTION : la base de données de l'ancienne base doit-être une version 0.9. Cependant sachez que la base que vous utilisez actuellement sera sauvegardées dans le dossier 'save' du répertoire de l'add-on)
Tout d'abord cliquez sur la liste des 'Utilitaires' puis sur 'Migrer une base de donnees (V1 -> V2)' dans l'interface du ShaderTools comme ici :
Une nouvelle fenêtre s'ouvre, il vous suffit de sélectionner votre ancienne base de matériaux comme ici :

Pendant la migration de la base de matériaux il vous sera impossible d'accéder aux modules 'Ouvrir...' ou encore 'Sauvegarder', comme indiqué ici :
Une fois la migration terminée, la base redeviendra accessible. Sachez tout de même qu'en cas de problème une copie de la base que vous utilisiez avant la migration est sauvegarder dans le dossier 'save' du ShaderTools.
