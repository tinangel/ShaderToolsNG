Le Module Ouvrir... :
Ce module vous permet d'accéder à votre base de matériaux pré-configurés et les intégrer dans votre scène.

Une nouvelle fenêtre s'ouvre, sélectionnez dans le panneau 'Bookmarks' (panneau à gauche de cette nouvelle fenêtre) le bookmark 'Base Materiaux', comme suit :
